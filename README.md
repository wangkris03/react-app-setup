# 从 0-1 搭建 react 工程，学习工程化

这个是一个 React + Vite + tailwindcss + jest 的前端项目，支持 SSR。

## Feature

- 开箱 typescript
- 支持 tailwind-jit
- eslint + prettier + husky
- jest + esbuild
